### コミットメッセージのテンプレート

以下のフォーマットに従ってコミットメッセージを書いてください。

```
[プレフィックス]: [変更内容の要約]

#### 記載例

```

feat: ユーザー登録APIを追加

fix: 食事登録時のカロリー計算バグを修正

docs: READMEにAPI一覧を追加

```

#### プレフィックス一覧

| プレフィックス | 用途                                   |
| -------------- | -------------------------------------- |
| feat:          | 新機能追加                             |
| fix:           | バグ修正                               |
| docs:          | ドキュメント変更                       |
| refactor:      | リファクタリング                       |
| chore:         | 設定変更など                           |
| test:          | テスト追加・修正                       |
| style:         | コードスタイル修正（動作に影響しない） |
```
